# Générateur de mots de passe

Générateur et gestionnaire de mots de passe en Python pur.

## Description

Ce programme génère des mots de passe aléatoires sécurisés et les
sauvegarde localement dans un fichier JSON associés au nom du site.
Il permet aussi de rechercher, afficher et supprimer des mots de passe.

## Fonctionnalités

- Générer un mot de passe aléatoire (majuscules, minuscules, chiffres, symboles)
- Sauvegarder un mot de passe associé à un site
- Afficher tous les mots de passe sauvegardés
- Rechercher un mot de passe par nom de site
- Supprimer un mot de passe

## Utilisation

```bash
python main.py
```
